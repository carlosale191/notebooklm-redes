# RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS

Data: 15/04/2026
Empresa: Abstergo Industries
Responsável: Carlos

## Introdução

Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por Carlos. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto

O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos específicos. A seguir, serão descritas as etapas do projeto:

Etapa 1:

- Serviço: Amazon EC2 (Elastic Compute Cloud)
- Foco: Fornecer capacidade computacional escalável na nuvem.
- Descrição: Com o EC2, é possível configurar instâncias que aumentam ou diminuem de tamanho automaticamente (Auto Scaling) conforme demanda. Solução para hospedagem de aplicações legadas ou servidores de aplicação customizados, garantindo que se pague conforme demanda de recursos usadas no volume de demandas atendidas.

Etapa 2:

- Serviço: Amazon S3 (Simple Storage Service)
- Foco: Armazenamento de objetos com alta durabilidade, escalabilidade e segurança.
- Descrição: Em um cenário corporativo, o S3 seria utilizado para armazenar todos os documentos PDF, imagens de produtos e vídeos de treinamento da empresa, além de servir como um repositório central para Logs de auditoria. Ele funciona como um "balde" infinito onde os arquivos ficam protegidos contra perda de dados (com durabilidade de 99,9%).

Etapa 3:

- Serviço: Amazon RDS (Relational Database Service)
- Foco:Serviço gerenciado para bancos de dados relacionais, facilitando a configuração, operação e escalabilidade.
- Descrição: Ideal para sustentar o sistema de ERP ou banco de dados de vendas. Ao invés da equipe perder horas configurando backups manuais ou atualizações de segurança no banco de dados, o RDS automatiza essas tarefas. Suporta motores populares como MySQL, PostgreSQL e SQL Server, permitindo que a empresa foque na análise dos dados e não na manutenção do servidor.

## Conclusão

A implementação de ferramentas na empresa Abstergo Industries tem como esperado economia de custo com infraestrutura, assim como economia de tempo relacionado a manutenções eventuais, configurações e backups, o que aumentará a eficiência e a produtividade da empresa. Recomenda-se a continuidade da utilizaçãoo das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
